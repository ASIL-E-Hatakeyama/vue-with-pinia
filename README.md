# vue-with-pinia

Vuex じゃなくて Pinia を使った状態管理の使い方サンプル
ログイン画面も作ってみて、API を呼んでどうにかするっていう感じの実装にしてみた。
なお ID/Pass を同じの入れればログインできる。

## Project setup

```
npm install
```
